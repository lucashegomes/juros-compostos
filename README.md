# Calculadora de Juros Compostos

Este projeto é uma calculadora de juros compostos simples, desenvolvida com HTML, CSS e JavaScript. Ele permite calcular o crescimento de um investimento ao longo do tempo, considerando valor inicial, taxa de juros, aportes regulares e período de tempo.

## Demonstração

Acesse a calculadora online: [Calculadora de Juros Compostos](https://lucashegomes.github.io/juros-compostos/)

## Funcionalidades

- Cálculo de juros compostos com base em:
  - Valor inicial
  - Taxa de juros (mensal ou anual)
  - Valor de aporte
  - Tempo de aporte (mensal ou anual)
- Exibição dos resultados de forma paginada.

## Como Usar

1. Insira os valores nos campos fornecidos:
   - Valor inicial
   - Taxa de juros
   - Valor do aporte
   - Tempo de aporte
   - Tipo de tempo (mensal ou anual)
   - Tipo de taxa (mensal ou anual)
2. Clique no botão **Calcular**.
3. Veja os resultados exibidos na tela, com a possibilidade de navegar entre as páginas.
